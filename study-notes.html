Study notes

I come across reference grammars of many languages, and here are some notes that I have made, presented in a way somewhat different from the original source. I hope that these help you too, in addition to consulting the original text. n.b.: I have not necessarily read the whole text before making these notes.

Georgian

Verbal indexing

There are two sets of verbal index affixes in Georgian. (i.e. 'cross-referencing'; an agreeing noun phrase does not need to occur.) I shall call them Set I and Set II. They are often called 'subject' and 'object' affixes respectively, but I shall refrain from these terms as Set II can also be used to mark subject.

Set I singular plural
1st person ვ- ვ-[...]-თ
v- v-[...]-t
2nd person ∅(/ხ-) ∅(/ხ-)[...]-თ
∅(/x-) ∅(/x-)[...]-t
3rd person -ს/-ა/-ო -ან/-ენ/-ნენ/-ეს
-s/-a/-o -an/-en/-nen/-es
Set II singular plural
1st person მ- გვ-
m- gv-
2nd person გ- გ-[...]-თ
g- g-[...]-t
3rd person ∅(/ს-/ჰ-) ∅(/ს-/ჰ-)[...](-თ)
∅(/s-/h-) ∅(/s-/h-)[...](-t)

There are complications when both Set I and Set II affixes appear on a verb; (amongst many complications,) there can only be one (overt) prefix and one (overt) suffix on a verb. When there is a conflict, only one of each is kept.
