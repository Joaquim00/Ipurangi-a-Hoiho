Study notes

I come across reference grammars of many languages, and here are some notes that I have made, presented in ways that help me understand better. I hope that these help you too, in addition to consulting the original text. n.b.: I have not necessarily read the whole text before making these notes.

Georgian

Verbal indexing

There are two sets of verbal index affixes in Georgian ('cross-referencing'; an agreeing noun phrase does not need to occur). I shall call these Set I and Set II. They are often called 'subject' and 'object' affixes respectively, but I shall refrain from using these terms as Set II can also be used to mark subjects in some cases. See Wikipedia for the position of these affixes in relation to other verbal affixes.

1st person ვ- ვ-[...]-თ
v- v-[...]-t
2nd person ∅(/ხ-) ∅(/ხ-)[...]-თ
∅(/x-) ∅(/x-)[...]-t
3rd person -ს/-ა/-ო -ან/-ენ/-ნენ/-ეს
-s/-a/-o -an/-en/-nen/-es
Set II singular plural
1st person მ- გვ-
m- gv-
2nd person გ- გ-[...]-თ
g- g-[...]-t
3rd person ∅(/ს-/ჰ-) ∅(/ს-/ჰ-)[...](-თ)
∅(/s-/h-) ∅(/s-/h-)[...](-t)

In the tables above, affixes separated by a slash are are allomorphs which are conditioned by other morphemes in the verb. Affixes in parentheses are rarer allomorphs:

  • Set I second person ხ- x-:
    • the present indicative copulas ხარ xar 'you(sg) are' and ხართ xart 'you(pl) are'; the present indicative copula is also often used as an auxiliary verb
    • the motion verb in the future and aorist series: future მო-ხ-ვალ-(თ) mo-x-val(-t) 'you will come', conditional მო-ხ-ვიდ-ოდ-ი(-თ) mo-x-vid-od-i(-t), future subjunctive მო-ხ-ვიდ-ოდ-ე(-თ) mo-x-vid-od-e(-t); aorist მო-ხ-ვედ-ი(-თ) mo-x-ved-i(-t) 'you came', aorist subjunctive (optative) მო-ხ-ვიდ-ე(-თ) mo-x-vid-e(-t). The suffix -თ -t signifies plural subject. The preverb მო- mo- can be substituted by other preverbs, e.g. მო-ხ-ვალ mi-x-val 'you will go'
  • Set II third person ს-/ჰ- s-/h-:
    • for direct object, ∅
    • for indirect object:
      • ს- s- in front of ც- წ- ძ- ჩ- ჭ- ჯ- თ- ტ- დ- ts- ts'- dz- tš- tš'- dž- t- t'- d- (coronal plosives and affricates)
      • ჰ- h- in front of ქ- კ- გ- ყ- k- k'- g- q'- (dorsal plosives) and პ- p'-
      • ∅ elsewhere; ს-/ჰ- s-/h- also becomes ∅ when the Set I first person ვ- v- occurs
  • Set II plural suffix -თ -t in third person: plurality is usually not shown with third person in set II (cf. below). The exception is in the perfect series; in the perfect series, the subject is indexed by a set II affixes, and the object is indexed by a copula auxiliary. The plural suffix -თ -t is used for such set II subjects. (What is called 'perfect' is primarily inferred/non-visual evidential; however, the pluperfect is an actual past-of-past)

There are complications when both Set I and Set II affixes appear on a verb. One complication is that there can be maximally one (overt) prefix and one (overt) suffix. When more than one prefix or more than one suffix is required, only one is kept (in the list below, > signifies 'has precedence over'):

  • Prefixes: Set II 2SG გ- g- > Set I 1SG ვ- v- (e.g. ვ-გ-მალ-ავ v-g-mal-av 'I hide you(sg)')
  • Suffixes: Set I 3PL -ან/-ენ/-ნენ/-ეს -an/-en/-nen/-es > Set II PL -თ -t > Set I 3SG -ს/-ა/-ო -s/-a/-o (e.g. გ-მალ-ავ-ენ-თ g-mal-av-en-t 'they hide you(pl)', გ-მალ-ავ-ს-თ g-mal-av-s-t 's/he hides you(pl)')
  • two plural suffixes -თ -t becomes one (e.g. ვ-გ-მალ-ავ-თ-თ v-g-mal-av-t-t 'we hide you(pl.)')

These deletion rules create two homonym sets, e.g. გმალავთ gmalavt 'we hide you(sg)', 'I hide you(pl)', 's/he hides you(pl)', or 'we hid you(pl)'; გმალავენ gmalaven 'they hide you(sg)' or 'they hide you(pl)'. In addition, for Set II third person plural, plurality is rarely shown, so the forms for set II third person plural is most usually the same as third person singular. In the table below, მალ mal is the verb root 'hide', and -ავ -av is a 'thematic suffix' (which is used primarily in the present/future series; in this case the verbs are in present indicative). Set I indexes the subject, and Set II the object, so, for instance, Set I 2SG and Set II 1sg მმალავ mmalav means 'You(sg) hide me'.
